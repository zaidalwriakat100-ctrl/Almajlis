# مجلد التفريغ الحرفي للجلسات

هذا المجلد مخصص لرفع ملفات التفريغ الحرفي (النصوص الكاملة) للجلسات البرلمانية.

## تنسيق الملفات
- **الصيغة**: ملفات نصية (.txt) أو JSON (.json)
- **التسمية**: يفضل تسمية الملف باسم الجلسة، مثال:
  - `الجلسة_الأولى.txt`
  - `الجلسة_الثالثة_اليوم_الأول.txt`
  - `session_1.json`

## هيكل الملف المقترح (JSON)
```json
{
  "session_id": "session_1",
  "session_title": "الجلسة الأولى",
  "date": "2024-11-18",
  "term": "الدورة العادية الأولى",
  "full_transcript": "النص الكامل للجلسة هنا..."
}
```

## أو ملف نصي بسيط (.txt)
```
الجلسة: الجلسة الأولى
التاريخ: 2024-11-18
الدورة: الدورة العادية الأولى

--- بداية التفريغ ---

[النص الكامل للجلسة]
```

## ملاحظات
- سيتم استخدام هذه الملفات كمصدر للبحث في تبويب "القضايا والمواضيع"
- كلما كان التفريغ أكثر دقة، كانت نتائج البحث أفضل
