🚀 خطوات التحديث:
═══════════════════════════════════════════════════════════════════════

الخطوة 1️⃣: ضع ملف .txt في المجلد
   📂 session_updates/raw_text/

الخطوة 2️⃣: شغّل سكريبت التحويل
   > cd frontend/public/data
   > node convert_txt_to_json.js

الخطوة 3️⃣: شغّل سكريبت التحديث
   > node update_sessions.js

✅ تم! البيانات الآن محدثة في sessions.json
